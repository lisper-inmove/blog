* 常用函数列表
#+begin_verse
1. count(start, step=1): 从start开始的无限数字.step表示步长
2. cycle(p): p是一个序列,返回从p[0]到p[-1],然后又从头开始
3. repeat(elem [,n]): n是可选参数,如果没有n则一直重复下去,有n的话则只重复n次
4. chain(p0, p1, ...): 迭代多个序列
#+end_verse

#+begin_verse
compress(sequence, selectors): 如果selectors[i]为真,那么sequence[i]被放在返回队列里
#+end_verse
#+NAME: compress-example
#+begin_src python :results value code :wrap SRC json :exports both
  import itertools

  lst = [1, 2, 3, 4]
  selectors = [1, 0, 0, 1]
  ret = itertools.compress(lst, selectors)

  ret_value = []
  for r in ret:
      ret_value.append(r)
  return ret_value
#+end_src

#+RESULTS:
#+begin_SRC json
[1, 4]
#+end_SRC

#+begin_verse
dropwhile(pred, seq): 用seq中的元素,依次调用pred.当有一个返回为False时,返回从此元素开始的所有元素
#+end_verse
#+NAME: dropwhile-example
#+begin_src python :results value code :wrap SRC json :exports both
  import itertools

  lst = [1, 3, 6, 4, 3]
  pred = lambda x: x < 5
  ret = itertools.dropwhile(pred, lst)
  ret_value = []
  for r in ret:
      ret_value.append(r)
  return ret_value
#+end_src

#+RESULTS: example-dropwhile
#+begin_SRC json
[6, 4, 3]
#+end_SRC

#+begin_verse
ifilter(pred, seq): 返回使pred为真的元素列表.pred是一个单参数函数
#+end_verse
#+NAME: ifilter-example
#+begin_src python :results value code :wrap SRC json :exports both
  import itertools

  lst = [1, 3, 4, 5]
  pred = lambda x: x + 1 == 6
  ret = itertools.ifilter(pred, lst)
  ret_value = []
  for r in ret:
      ret_value.append(r)
  return ret_value
#+end_src

#+RESULTS: ifilter-example
#+begin_SRC json
[5]
#+end_SRC
