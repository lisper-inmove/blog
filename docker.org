#+TITLE: docker

* Ubuntu 安装 Docker
[[https://docs.docker.com/engine/install/ubuntu/][官网地址]]
1. 删除旧版本的docker
   #+BEGIN_SRC shell
     sudo apt-get remove docker docker-engine docker.io containerd runc
   #+END_SRC
2. 设置 repository
   #+BEGIN_SRC shell
     sudo apt-get update
     sudo apt-get install \
          apt-transport-https \
          ca-certificates \
          curl \
          gnupg \
          lsb-release
   #+END_SRC
3. 设置官方的 GPG key
   #+BEGIN_SRC shell
     curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /usr/share/keyrings/docker-archive-keyring.gpg
   #+END_SRC
4. 设置官方的 stable repo
   #+BEGIN_VERSE
   把stable替换成 *nightly* 或者 *test* 来安装其它处本的docker
   #+END_VERSE

   x86_64/amd64
   #+BEGIN_SRC shell
      echo \
       "deb [arch=amd64 signed-by=/usr/share/keyrings/docker-archive-keyring.gpg] https://download.docker.com/linux/ubuntu \
       $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null
   #+END_SRC

   armhf
   #+BEGIN_SRC shell
     echo \
       "deb [arch=armhf signed-by=/usr/share/keyrings/docker-archive-keyring.gpg] https://download.docker.com/linux/ubuntu \
       $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null
   #+END_SRC

   arm64
   #+BEGIN_SRC shell
     echo \
       "deb [arch=arm64 signed-by=/usr/share/keyrings/docker-archive-keyring.gpg] https://download.docker.com/linux/ubuntu \
       $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null
   #+END_SRC

   #+BEGIN_VERSE
   注意 $(lsb_release -cs) 有可能不在列表中 第5步 apt-get update的时候就会报404的错误.这个时候去 https://download.docker.com/linux/ubuntu
   去找一个存在的替换掉即可.但是要注意要和你的系统契合.
   #+END_VERSE
5. 更新 apt 包索引并安装最新版本的docker
   #+BEGIN_SRC shell
     sudo apt-get update && sudo apt-get install docker-ce docker-ce-cli containerd.io
   #+END_SRC
* docker commit 来提交修改
#+BEGIN_EXAMPLE
  $ docker run -i -t ubuntu
  root@739869270af3:/# apt update
  $ docker commit container-id ubuntu:update
  $ docker images
#+END_EXAMPLE
* 从容器中导出tar以及导入成镜像
#+BEGIN_EXAMPLE
  $ docker export container-id > xxx.tar
  $ docker import - image-name < xxx.tar
#+END_EXAMPLE
* 从镜像中导出成tar以及导入成镜像
#+BEGIN_EXAMPLE
  $ docker save -o xxx.tar image-name
  $ docker load < xxx.tar
#+END_EXAMPLE
