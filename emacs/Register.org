#+SETUPFILE: ~/.emacs.d/org-templates/template.org
#+TITLE: 注册器: Register


* Register的作用
#+begin_verse
Register的作用是记录下当前的位置.后面可以通过快捷键迅速跳转回这个位置
#+end_verse
* 相关快捷键
#+begin_verse
1. C-x r SPC c: 把当前position记录到注册器c
2. C-x r j c: 跳转到注册器c
3. M-x view-register RET c: 查看注册器c的内容
4. C-x r s c: 把选中区域放到注册器c中
5. C-x r i c: 把注册器c中的内容插入到当前位置
5. C-x r r c: 把选中的矩形存到注册器c中
#+end_verse
