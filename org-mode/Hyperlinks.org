#+begin_verse
当我们需要在Org文件中插入超链接时,可以使用以下形式直接直接文本输入

#+end_example

** 如何生成内部链接这些都不要去记它
#+begin_verse
只要是创建org文件内部的链接的时候,用 =org-store-link= 函数
和 =org-insert-link= 这两个函数就可以了

用 =org-store-link= 函数生成链接
然后用 C-c C-l (=org-insert-link=) 来插入这个链接.
=org-store-link= 在生成链接的时候
1. 如果当前光标下是 =<<target>>= 的形式,则链接指向此target
2. 如果这个提纲有CUSTOM_ID属性,则链接指向此custom ID
3. 以上两点都没有,生成提纲链接
#+end_verse

** Org文件中链接之间的跳转
1. C-c C-o: 跟随链接.跳转到link对应的位置
2. C-c &: 链接回跳.因为 =C-c &= 被yasnippet占用了.所以我绑定到了 =C-o C-c=
#+begin_verse
当我们 /跟随链接/ 的时候,OrgMode会把当前位置放到一个mark ring中
然后当我们回跳的时候,就可以根据这个mark ring往回跳了
#+end_verse

* 快捷键
1. C-c C-l (org-insert-link): 添加或者修改链接
2. C-u C-c C-l: 添加或者修改文件链接
3. C-c C-o (org-open-at-point): 打开当前链接
4. C-c % (org-mark-ring-push): 把当前位置放到Org mark ring.
5. C-c & (org-mark-ring-goto): 回跳到前一个Org mark ring.因与yasnippet快捷键冲突,已被映射到 C-o C-c
6. C-c C-x C-n/p (org-next-link) (org-previous-link)
