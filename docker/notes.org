#+SETUPFILE: ~/.emacs.d/org-templates/template.org
#+TITLE: Docker Usage

* Image
1. List Images
   #+begin_src shell
     sudo docker images
   #+end_src
2. Pull Image
   #+begin_src shell
     docker pull repo
   #+end_src

* Container
1. Run a Container
   1. -d: Run as daemon
   2. --rm: Remove container when stop the container
   3. -p: Mapping the port 1780 of container to 1790 of the host machine.
   4. --name: Unique name of the container.
   #+begin_src shell
     sudo docker run -d --rm -p 1790:1780 -p 1100:1100 --name container-name image-id
   #+end_src
2. Stop a Container
   #+begin_src shell
     sudo docker stop container-id
   #+end_src
3. Attach a Container
   #+begin_src shell
     sudo docker exec -i -t container-id /bin/bash
   #+end_src
4. List Containers
   1. -a: List all containers, include exited containers.
   #+begin_src shell
     sudo docker ps -a
   #+end_src
