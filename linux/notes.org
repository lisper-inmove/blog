* Set generate location of core dump file
#+begin_src shell
  echo "/home/inmove/corefile/core-%e-%p-%t" > /proc/sys/kernel/core_pattern
#+end_src
Enusre that corefile exists and process effective IDs can write in it.
1. %p: pid or the process
2. %u: real uid
3. %g real gid
4. %s: number of singal causing dump
5. %d: time of dump
6. %h: hostname
7. %e: executable file
8. %E: pathname of executable, with slashes('/') replaced by exclamation marks ('!')
9. %c: core file size soft resource limit of crashing process
