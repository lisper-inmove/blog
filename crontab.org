* 基本形式

  #+BEGIN_VERSE
  * * * * * Command
  1. 分钟
  2. 小时
  3. 日
  4. 月
  5. 星期
  6. 要执行的命令
  #+END_VERSE

* 实例

  #+BEGIN_EXAMPLE
  
  1. 每分钟执行
  * * * * * Command
  2. 每小时的第3,5分钟执行
  3,5 * * * * Command
  3. 每天10点到12点第3,5分钟执行
  3,5 10-12 * * * Command
  4. 每隔3天的上午10点到12点的3,5分钟执行
  3,5 10-12 */3 * * Command
  5. 每周一上午10点到12点的3,5分钟执行
  3,5 10-12 * * 1 Command
  6. 每月1,11,21,31号执行
  * * 1,11,21,31 * * Command

  #+END_EXAMPLE

* 注意事项

  #+BEGIN_VERSE
  
  1. Crontab不会加载环境变量
  2. 使用全局路径

  #+END_VERSE
